## 包含模块 
* site/web
    提供后台服务，java 编写，spring boot 框架，maven 架构。
* site/html
    前端构建，视图层，逻辑层。
* site/app
    程序外壳，支持终端(web，ios，android) 
* module/cache
    redis 支持
* module/client
    httpClient（RestClient）支持
* module/email
    javaMail支持
* module/jms
    jms（activeMq）支持
* module/schedule
    schedule（quart）支持
*core
    base 基础工具类
*asset/angular
    angular 单页面应用 for app
*asset/sea
  s eajs 模块化加载 for web
